Welcome to TotalReCAL

A powerful iCal feed aggregator and filter with the following features:

  • Subscribe to multiple remote iCal feeds
  • Accumulate all events forever (never deleted)
  • Create filtered output feeds with custom rules
  • Filter by categories, regex patterns, and time overlap
  • Manually hide/show individual events

Quick Start

  1. Add a source - Subscribe to a remote iCal feed
  2. Create an output - Define a filtered feed
  3. Manage events - View and hide specific events
  4. Add auto-hide rules - Automatically hide overlapping events
